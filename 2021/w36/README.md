# Week 36 challenge

Write a function `findMinInt` which takes in an array of numbers and returns the minimum integer value.
Thus, non-integer numbers are ignored

Examples:
```
findMinInt([7, -4, 12.6, -21.32, -2, 0, 5]) // returns -4
```
