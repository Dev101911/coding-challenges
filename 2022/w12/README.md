# Week 12 challenge

Write a function `expandIt` which accepts a number and returns a string in Mathematical Expanded Notation


Examples:
```
expandIt(4) // returns '4'
expandIt(17) // returns '10 + 7'
expandIt(1024) // returns '1000 + 20 + 4'
```
